      - ## Getting Started
     
      - ### Prerequisites
      - - A modern web browser (Chrome, Firefox, Safari, Edge)
        - - No additional dependencies required
         
          - ### Installation
         
          - 1. Clone the repository:
            2. ```bash
               git clone https://github.com/gawilliamssr/Slider-mitt.git
               cd Slider-mitt
               ```

               2. Open `index.html` in your web browser or use a local server:
               3. ```bash
                  python -m http.server 8000
                  # Then navigate to http://localhost:8000
                  ```

                  ## Usage

                  Simply open the website and interact with the slider controls:
                  - Click the navigation buttons to move between slides
                  - - The slider will automatically cycle through content
                    - - All animations are smooth and responsive
